# viSTL

## Requirements

- [Python 2.7](https://www.python.org/downloads/)
- [Pipenv](https://github.com/pypa/pipenv)
    - `pip install pipenv`

### Additional Requirements for Windows

- [Microsoft Visual C++ 9.0](http://aka.ms/vcpython27)

## Installation

```bash
pipenv install
```

## Running

```bash
pipenv shell
python main.py -h
```
